# 🎌 Anime Recommender — Combined System

A unified anime recommendation app combining **two engines** into one Streamlit interface:

| Engine | Method | Source Repo |
|---|---|---|
| **Tab 1 — AI Recommender** | Groq LLM + ChromaDB + LangChain RAG | `anime_recommender` |
| **Tab 2 — Similarity Search** | TF-IDF + Cosine Similarity (scikit-learn) | `Anime-Recommendation-system` |

---

## ⚙️ Setup

### 1. Clone & Install

```bash
git clone https://github.com/Shrikant-Bawankule/anime-recommender-combined.git
cd anime-recommender-combined
pip install -r requirements.txt
```

### 2. Set Up Environment Variables

```bash
cp .env.example .env
# Edit .env and add your GROQ_API_KEY
```

### 3. Set Up Data Files

> ⚠️ See [`data/DATA_SETUP.md`](data/DATA_SETUP.md) for full instructions.

```bash
# Copy datasets from source repos:
cp ../anime_recommender/data/anime_with_synopsis.csv data/
cp ../anime_recommender/data/anime_updated.csv data/
cp ../Anime-Recommendation-system/anime.csv data/
```

### 4. Build ChromaDB Vector Store (Tab 1 only)

```bash
python pipeline/build_pipeline.py
```

This reads `data/anime_with_synopsis.csv`, processes it, and saves the vector store to `chroma_db/`.

### 5. Run the App

```bash
streamlit run app/app.py
```
